Week 1: Making a Website

For the first week's assignment, our task was to curate our own website for our work.


I started out by installing a few different items first: Visual Studio code, git using my terminal, and the brackets extension.

Next up was setting up the repository for my website through GitHub Pages. After some use of terminal, it was set up and ready for the next step.


Getting up the index.html took some time. It involved the uses of committing, staging changes, and pushing. Had a little bit of diffculty nagivating through this step, but was able to pull through.


Moving along, the next step was finding a good template to use. I went on to the W3 school's website and found this one.


I made changes to the template via the Tryit Editor on W3's site until they were to my liking.


Finally, I copied the template code and pasted it into my index.html file on VSC. Now just time to commit and push!
